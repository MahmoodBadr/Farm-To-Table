# Farm To Table Android App

**Technologies:** Java, Android Studio, Firebase, Google Maps API  

## Overview
The **Farm To Table** Android app connects local farmers with consumers, allowing users to browse and purchase fresh produce directly. The app features real-time delivery tracking and photo confirmation to ensure reliability and transparency.  

## Features
- **User Authentication:** Secure sign-up and login powered by Firebase  
- **Order Management:** Browse products, place orders, and view order history  
- **Payment Handling:** Integrated payment system for seamless transactions  
- **Delivery Tracking:** Real-time tracking of deliveries with photo confirmation on arrival  
- **UI/UX Improvements:** Two rounds of user testing to enhance readability, consistency, and overall user experience  
- **Maps Integration:** Google Maps API used for accurate location tracking and route display
